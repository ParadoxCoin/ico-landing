# Product Overview

## Project Purpose
AI SaaS Platform is a production-ready, multi-modal AI service platform that provides chat, image generation, video generation, and audio generation capabilities through a unified credit-based billing system. The platform enables users to access multiple AI providers through a single interface while managing their usage through a flexible credit system.

## Value Proposition
- **Unified AI Access**: Single platform for multiple AI modalities (chat, image, video, audio)
- **Credit-Based Flexibility**: Pay-as-you-go model with transparent pricing across all services
- **Multi-Provider Support**: Access to 40+ AI models from leading providers (Fireworks AI, OpenAI, Replicate, Pollo.ai, ElevenLabs, Fal.ai)
- **Enterprise-Ready**: Production-grade security, monitoring, and scalability features
- **Developer-Friendly**: Comprehensive API with 54 endpoints, full documentation, and API key management

## Key Features

### Core AI Services
- **Chat Generation**: AI-powered conversations with multiple model options (Fireworks AI, OpenAI)
- **Image Generation**: 25+ models from Fal.ai and Replicate for diverse image creation needs
- **Video Generation**: 43 Pollo.ai models with text-to-video and image-to-video capabilities, plus 21 video effects
- **Audio Generation**: Voice synthesis via ElevenLabs and Google TTS with voice cloning support
- **Synapse Agent**: Autonomous agent for complex multi-step tasks with real-time progress tracking

### User Management
- **Authentication**: JWT-based auth with OAuth support (Google, GitHub, Discord)
- **Profile Management**: User profiles with customizable settings and preferences
- **API Key System**: Generate and manage multiple API keys for programmatic access
- **Transaction History**: Complete audit trail of all credit usage and purchases
- **Usage Statistics**: Detailed analytics on service consumption and spending patterns

### Media Library
- **Organized Storage**: Separate galleries for images, videos, and audio files
- **2-Month Retention**: Automatic media storage with configurable retention policies
- **Showcase Feature**: Public gallery system for sharing generated content
- **Bulk Operations**: Multi-select, delete, and download capabilities
- **Storage Analytics**: Track storage usage and file statistics

### Billing & Credits
- **Flexible Packages**: Starter ($10/1,100 credits), Pro ($25/2,800 credits), Enterprise ($100/12,000 credits)
- **Multiple Payment Methods**: Credit card, cryptocurrency, Binance Pay, MetaMask (15% discount)
- **Dynamic Pricing**: Admin-configurable service costs per AI provider
- **Invoice System**: Automated invoice generation and download
- **Refund Support**: Built-in refund request and processing workflow

### Admin Panel
- **User Management**: View, edit, suspend, and manage user accounts
- **Credit Administration**: Manually adjust user credit balances with audit logging
- **Service Cost Control**: Configure pricing for each AI service and model
- **Platform Analytics**: Real-time statistics on users, revenue, and service usage
- **Package Management**: Create and modify pricing packages with discount tiers

### Security & Monitoring
- **Rate Limiting**: DoS protection with slowapi (per-user and per-IP limits)
- **Error Tracking**: Sentry integration for real-time error monitoring and alerting
- **Structured Logging**: Loguru-based logging with log levels and rotation
- **Role-Based Access**: User/Admin role separation with permission checks
- **Input Validation**: Pydantic schemas for all API requests and responses
